# Ip-Information
An Excellent OSINT tool to get information of any ip address. All details are explained in below screenshot.

# Important!

```
1. apt install python3
```
```

```
```
3. cd Ip-Information
```
```
4. ls
```
```
5. pip install -r requirements.txt
```
```

```
